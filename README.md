# Hospital-registration-and-appointment-system
这款医院挂号预约微信小程序，专为便捷就医打造。患者动动手指，随时随地就能查询医院科室、医生信息，依据需求挑选合适的就诊时间，一键完成挂号预约。就诊前，小程序贴心推送提醒，防止错过就诊。还支持在线支付挂号费，查看历史挂号与就诊记录。有了它，告别线下长时间排队，让就医流程更高效、轻松，极大提升患者就医体验 。
